### 数据质量说明

- **安装信息缺失 (7.4%)**：部分拆换记录仅有拆下信息，无对应的安装记录
- **拆下信息缺失 (8.9%)**：部分拆换记录仅有安装信息（如新件直接装机）
- **站点缺失 (6.0%)**：少量记录未记录拆换地点，站点字段为空
- **拆换原因缺失 (2.1%)**：少量记录未填写拆换原因描述
- **送修合同号 (58.6%)**：大部分拆下件未直接送修，可能进行了库存保留或其他处理

### 业务说明

- 每条记录代表一次完整的拆换操作（拆下 + 安装）
- 安装与拆下部件通过 CHID 关联
- RONO 字段表示拆下件最后一次送修的合同号，为空表示该件未送修或送修记录不在本数据中

- **站点匹配**：21章拆换记录中的 ZSZBM 字段可与本表的 CAIR 字段进行关联

- **完整度**：所有核心采购字段（订单号、件号、价格、数量等）完整，无缺失

### 业务说明

- **采购管理**：通过 `EBELN` 可以追踪每一笔采购订单的详细信息，包括供应商、价格、数量等

- **财务核算**：`NETPR` 和 `NETWR` 字段提供了成本信息，用于财务核算和成本控制
- **租赁业务**：`ZZJRRQ` 和 `ZZGHRQ` 记录租赁件的租借和归还日期
- **维修周期**：`ZXLTAT` 字段记录修理周期，可用于分析送修时效性
